# Security Policy

This policy covers **security vulnerabilities in the Wall & Shadow code**. For
reports of illegal or abusive **content** on the hosted service, use
`abuse@wallandshadow.com` via the in-app About section instead — not this
channel.

## Reporting a vulnerability

Please report privately, not in a public issue:

- **Preferred:** GitHub private vulnerability reporting — the **Security** tab >
  **Report a vulnerability**.
- **Alternative:** email **info@wallandshadow.com**.

## Supported versions

Wall & Shadow is a hosted service; the supported version is the one currently
deployed at https://wallandshadow.com.

## Response

This is a small, solo-maintained project, so responses are best-effort. Please
allow a reasonable period for a fix before any public disclosure.
