# Code of Conduct

> Brief summary — to be expanded later.

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1, as its code of conduct. Until the full text is reproduced here,
the Covenant's published text is authoritative.

In short: be respectful and constructive in all project spaces.

Report concerns to **info@wallandshadow.com**.
